Lab 4: Sorting Integers

-----------
DESCRIPTION

In this lab, the user enters hexidecmal numbers in the program arguments in 0x00 format, the program will print those hex numbers.
The program will than print those numbers unsorted in decimal format, after that it will print those number in sorted order.
