Lab 3: ASCII-risks (Asterisks)
-----------
DESCRIPTION

In this lab, the user inputs a number, to indicate the height of the 
variable-sized ASCII diamonds and a sequence of embedded numbers.

(MARS). Enter the test case as a program argument and run using MARS.
